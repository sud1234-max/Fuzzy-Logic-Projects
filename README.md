Integration of Fuzzy Logic and Neural Networks:
This project uses an ANFIS (Adaptive Neuro-Fuzzy Inference System). The Fuzzy Logic part provides the "If-Then" structure so we can understand the grading rules. The Neural Network part acts as the "learner," using backpropagation to look at student data and automatically adjust the membership functions (the triangles) so the predictions become as accurate as possible.
